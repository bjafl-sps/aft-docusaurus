# Verktøy og maler

Denne seksjonen dekker de konkrete verktøyene jobbkonsulenten bruker i
kartlegging, oppfølging og dokumentasjon.

| Verktøy | Bruksområde | Fase |
|---|---|---|
| [Karriereverktøy](./karriereverktoy.md) | RIASEC, interessekartlegging, yrkesutforsking | Fase 2 |
| [Livshjulet](./livshjulet.md) | Helhetsbilde av livssituasjonen | Fase 2, tidlig |
| [Systematisk observasjon](./observasjon.md) | Strukturert observasjon i arbeidssituasjoner | Fase 5 + 2.5 |
| [Grunnleggende ferdigheter](./grunnleggende-ferdigheter.md) | Kartlegging og styrking av lesing/skriving/regning | Fase 2–5 |
| [Dokumentasjonsmaler](./dokumentasjonsmaler.md) | Plan, praksisavtale, observasjonsskjema, sluttrapport | Hele løpet |

Verktøyene er hjelpemidler — ikke fasit. De åpner samtaler og strukturerer
observasjoner, men det er relasjonen og jobbkonsulentens skjønn som bærer
prosessen.
