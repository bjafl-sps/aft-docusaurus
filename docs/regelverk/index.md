# Regelverk og rammer

Tre formelle rammer styrer gjennomføringen av AFT:

| Tema | Hva det handler om |
|---|---|
| [Kravspesifikasjonen](./kravspesifikasjonen.md) | NAVs krav til innhold, kvalitet og resultater (19.12.2025) |
| [Varighet og deltakelse](./varighet-og-deltakelse.md) | Tidsrammer, 3-månedersfrist, deltakelsesprosent og tilskudd |
| [Personvern](./personvern.md) | Taushetsplikt, GDPR, samtykke og hva som kan deles med hvem |

:::note[Regelverksforankring]
AFT er hjemlet i forskrift om arbeidsmarkedstiltak kapittel 13.
Kravspesifikasjonen fra Nav Rogaland (19.12.2025) presiserer kravene
og gjelder foran eldre rutiner ved motstrid.
:::
