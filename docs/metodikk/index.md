# Metodikk

SPS' faglige metodikk består av seks tilnærminger som til sammen utgjør
det faglige fundamentet i AFT-arbeidet.

| Tilnærming | Rolle | Seksjon |
|---|---|---|
| **Supported Employment** | Strukturelt rammeverk for hele tiltaksløpet | [Les mer](./supported-employment.md) |
| **Motiverende intervju (MI)** | Primærrammeverk for all veiledning og relasjon | [Les mer](./motiverende-intervju.md) |
| **Styrkebasert tilnærming** | Gjennomgående perspektiv — ressurser fremfor mangler | [Les mer](./styrkebasert.md) |
| **Karriereveiledning** | Kontinuerlig prosess, særlig tyngde i fase 2 | [Les mer](./karriereveiledning-metode.md) |
| **GAP-modellen** | Tilrettelegging i arbeidssituasjoner | [Les mer](./gap-modellen.md) |
| **LØFT** | Løsningsfokusert tilnærming i samtaler | [Les mer](./loft.md) |

SE er det strukturelle rammeverket — det definerer arbeidsgangens logikk.
MI er grunnholdningen i all kontakt med deltaker. De øvrige er supplerende
perspektiver og verktøy som aktiveres etter situasjon og fase.
