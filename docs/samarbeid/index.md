# Samarbeid

Tre relasjoner er avgjørende for å lykkes med AFT:

| Partner | Rolle | Primær kontakt |
|---|---|---|
| **Nav-veileder** | Oppdragsgiver, planeier, koordinator | Jobbkonsulent + avdelingsleder |
| **Arbeidsgiver** | Arena for ordinær arbeidstrening og ansettelse | Jobbkonsulent |
| **Andre aktører** | Helsetjenester, opplæring, kommunale tjenester | Etter behov og samtykke |

Jobbkonsulenten er bindeleddet mellom alle partene — og deltaker er alltid
informert om hva som deles med hvem.

Se undersidene for detaljer om hvert samarbeidsforhold.
