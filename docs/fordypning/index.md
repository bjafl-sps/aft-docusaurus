# Fordypning

Denne seksjonen gir teoretisk grunnlag for deg som vil forstå dypere —
ikke bare *hva* vi gjør, men *hvorfor* det virker.

De fire temaene belyser sentrale utfordringer i AFT-målgruppen og gir faglig
forankring for metodisk praksis:

| Tema | Relevans for AFT |
|---|---|
| [Psykisk helse](./psykisk-helse.md) | Funksjonsrettet perspektiv, IPS-prinsipper, jobbkonsulentens grenser |
| [Autistisk utbrenthet og masking](./autistisk-utbrenthet.md) | Hva skjules i ordinære arbeidsmiljøer — og hva det koster |
| [Epistemic trust](./epistemic-trust.md) | Tillit som forutsetning for læring og endring (Fonagy/AMBIT) |
| [Sosialt arbeid som fag](./sosialt-arbeid.md) | Person-i-situasjon, makt og brukerperspektivet |

:::info[Fordypning]
Innholdet her er beregnet på erfarne jobbkonsulenter eller ved faglig fordypning.
Det gir ikke instruksjoner om hva du skal gjøre — men faglig grunnlag for å
forstå situasjoner og ta velfunderte vurderinger.
:::
