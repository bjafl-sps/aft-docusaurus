Praktiske regler for Docusaurus v3 i dette prosjektet basert på feil og løsninger fra Fase 1+2.

**Why:** Disse mønstrene er ikke åpenbare fra dokumentasjonen og er alle blitt oppdaget via bygge- eller kjøretidsfeil i prosjektet.

**How to apply:** Slå opp her før du gjør strukturelle endringer i sidebar, MDX-filer eller kategorier.

## Sidebar — autogenerering

`_category_.json` fungerer KUN når Docusaurus scanner katalogen som en underkatalog under `dirName: '.'`.

**Riktig** (i `sidebars.ts`):
```ts
mainSidebar: [{type: 'autogenerated', dirName: '.'}]
```

**Feil** (gjør sidebaren flat, ingen kollapsbare kategorier):
```ts
mainSidebar: [
  {type: 'autogenerated', dirName: 'tiltakslopet'},
  {type: 'autogenerated', dirName: 'metodikk'},
  // ...
]
```

## `_category_.json`-format

```json
{
  "position": 3,
  "label": "Tiltaksløpet",
  "collapsible": true,
  "collapsed": false,
  "link": {
    "type": "doc",
    "id": "tiltakslopet/index"
  }
}
```
`metodikk/_category_.json` mangler `collapsible`/`collapsed` med vilje (Bjarte fjernet dem) — ikke legg dem tilbake.

## Skjule innhold

- **`_`-prefix på katalog/fil**: Ekskluderer fra autogenerering (brukt for `_fordypning/`)
- **`unlisted: true` i frontmatter**: Skjuler fra sidebar og søk, men URL er fortsatt tilgjengelig (brukt for `docs/superpowers/plans/`)

## Admonition-titler

**Riktig format** (klammeparenteser):
```mdx
:::note[Regelverksforankring]
:::tip[Praktisk tips]
:::warning[Vær oppmerksom]
:::info[Faglig kontekst]
```

**Feil** (uten klammer — vises ikke riktig):
```mdx
:::note Regelverksforankring
```

## MDX i `.md`-filer

Docusaurus v3 behandler `.md`-filer som MDX som standard. React-importer og JSX virker i `.md`-filer — du trenger ikke alltid omdøpe til `.mdx`. Unntak: bruk `.mdx` eksplisitt når filen bruker React-importer og andre filer lenker til den med filnavn.

## Broken links i footer/navbar

Footer-lenker til kategori-rot fungerer ikke med bare `/docs/metodikk` — Docusaurus klager ved bygg. Bruk alltid spesifikk doc-sti: `/docs/metodikk/supported-employment`.

## Kategori-index-sider

`index.md` (eller `index.mdx`) i en katalog blir automatisk kategoriens klikk-side i autogenerert sidebar. Lenk til den via `_category_.json` med `"type": "doc", "id": "katalognavn/index"`.
